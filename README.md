# Monico Maglapid Portfolio

This is a Laravel-based portfolio application for Monico Maglapid.

## Features

- Dynamic Hero Section
- About Me Section with Skills Progress Bars
- Experience Timeline
- Work/Portfolio Gallery
- Contact Information

## Setup

1. Clone the repository.
2. Run `composer install`.
3. Copy `.env.example` to `.env` and configure your database.
4. Run `php artisan key:generate`.
5. Run `php artisan migrate --seed` (to seed the database with Monico's information).
6. Run `npm install && npm run build`.
7. Run `php artisan serve`.
